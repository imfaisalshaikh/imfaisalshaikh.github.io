Learnify — Micro Learning Platform

Role: FlutterFlow Developer, Firebase Architect, UI/UX Designer
Stack: FlutterFlow (Flutter), Firebase Auth, Firestore, Firebase Storage

Overview

Learnify is a micro‑learning platform designed for fast, focused learning. Users can take short lessons, complete quizzes, track progress, and earn badges. It’s built for students, professionals, and anyone who wants to learn in small, effective sessions.

Problem

Traditional learning platforms are overwhelming, long, and time‑consuming. Modern learners prefer short, actionable lessons they can complete in minutes — not hours. There is a need for a simple, mobile‑friendly micro‑learning experience.

Solution

Learnify breaks learning into bite‑sized lessons with quizzes, progress tracking, and badges. The platform is optimized for mobile and built using FlutterFlow + Firebase for real‑time updates.

Key Features

  • Short lessons with text, images, and videos.
  • Quizzes with multiple‑choice questions and instant scoring.
  • Progress tracking for each course and lesson.
  • Badges & achievements for completing modules.
  • Course categories (Tech, Business, Productivity, Skills).
  • Search & filters for fast discovery.
  • Cross‑platform (iOS, Android, Web) using FlutterFlow.

FlutterFlow Implementation

  • Course list page: Firestore query with category filters.
  • Lesson viewer: Supports text, images, and embedded videos.
  • Quiz engine: Multiple‑choice questions with instant scoring.
  • Progress tracking: Writes to user_progress collection.
  • Badges: Auto‑assigned when course completion reaches 100%.
  • Responsive UI: Works on mobile and web.

My Role & Responsibilities

  • Designed the entire learning flow and UX.
  • Built all screens in FlutterFlow with responsive layouts.
  • Created Firestore schema and security rules.
  • Implemented quiz logic and scoring system.
  • Developed progress tracking and badge automation.
  • Connected all pages to Firebase backend.

Outcome

Learnify demonstrates my ability to build structured, educational platforms using FlutterFlow + Firebase. It showcases quizzes, progress tracking, badges, and dynamic content — making it a strong portfolio project for clients looking for LMS, e‑learning, or training apps.
