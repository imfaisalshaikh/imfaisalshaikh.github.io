MiniMeet — Community Events & Meetups App

Role: FlutterFlow Developer, Firebase Architect, UI/UX Designer
Stack: FlutterFlow (Flutter), Firebase Auth, Firestore, Firebase Storage, Cloud Functions

Overview

MiniMeet is a cross‑platform events and meetups app that helps users discover, create, and join local events. Whether it's a workshop, sports meetup, study group, or community gathering, MiniMeet makes it easy to connect people with shared interests.

Problem

People often want to join local activities but struggle to find relevant events nearby. Existing platforms are either too complex, too broad, or not community‑focused. Small groups and local organizers need a simple way to create and promote meetups.

Solution

MiniMeet provides a clean, simple, and real‑time event discovery experience. Users can browse events by category, location, and date, join meetups instantly, and chat with other attendees. Organizers can create events in minutes.

Key Features

  • Event discovery with filters (category, date, location).
  • Create events with images, descriptions, and attendee limits.
  • Join/Leave events with real‑time attendee updates.
  • Event chat for attendees to communicate.
  • Organizer dashboard to manage events and attendees.
  • Push notifications for event reminders and updates.
  • Cross‑platform (iOS, Android, Web) using FlutterFlow.

FlutterFlow Implementation

  • Event list page: Firestore query with filters and sorting.
  • Event details page: Dynamic page showing event info + attendees.
  • Join event flow: Writes to event_attendees collection.
  • Organizer tools: Edit event, delete event, view attendees.
  • Chat system: Real‑time messages using Firestore streams.
  • Push notifications: Cloud Functions + FCM for reminders.
  • Responsive UI: Works on mobile and web.

My Role & Responsibilities

  • Designed the full UX for event discovery and creation.
  • Built all screens in FlutterFlow with responsive layouts.
  • Created Firestore schema and security rules.
  • Implemented real‑time chat and attendee updates.
  • Integrated push notifications for event reminders.
  • Tested the app across mobile and web platforms.

Outcome

MiniMeet demonstrates my ability to build real‑time, community‑driven apps using FlutterFlow + Firebase. It showcases event creation, chat, notifications, and dynamic Firestore queries — making it a strong portfolio project for clients looking for social or community apps.
