TrustBox — Client Testimonial Collector

Role: FlutterFlow Developer, Firebase Architect, UI/UX Designer
Stack: FlutterFlow (Flutter), Firebase Auth, Firestore, Firebase Storage

Overview

TrustBox is a testimonial collection and management tool designed for freelancers, agencies, and small businesses. It allows clients to submit testimonials, which the owner can approve, reject, or feature publicly. It’s a simple but powerful system that builds trust and boosts conversions.

Problem

Freelancers and businesses struggle to collect testimonials consistently. Clients often forget, forms are scattered, and there’s no easy way to manage or display testimonials professionally.

Solution

TrustBox provides a clean, centralized system where: clients submit testimonials → owner reviews → approved testimonials appear publicly. It’s simple, fast, and built for real‑world use.

Key Features

  • Public testimonial form for clients to submit feedback.
  • Admin dashboard to approve, reject, or edit testimonials.
  • Featured testimonials for portfolio or landing pages.
  • Star ratings (1–5 stars).
  • Client photos (optional upload).
  • Public testimonial wall with filters.
  • Cross‑platform (iOS, Android, Web) using FlutterFlow.

FlutterFlow Implementation

  • Public form: Clients submit name, rating, message, and optional photo.
  • Admin dashboard: Approve, reject, or feature testimonials.
  • Public testimonial wall: Shows only approved testimonials.
  • Featured section: Displays highlighted testimonials on landing pages.
  • Firestore rules: Only admins can approve or modify testimonials.
  • Responsive UI: Works on mobile and web.

My Role & Responsibilities

  • Designed the full testimonial submission and approval flow.
  • Built all UI screens in FlutterFlow.
  • Created Firestore schema and admin‑only security rules.
  • Implemented approval, rejection, and featured logic.
  • Developed the public testimonial wall.
  • Connected all pages to Firebase backend.

Outcome

TrustBox demonstrates my ability to build business‑focused tools with admin workflows, moderation logic, and public‑facing components. It’s a strong portfolio project for clients looking for CRM tools, feedback systems, or internal dashboards.
